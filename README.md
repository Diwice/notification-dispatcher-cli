# Notification Dispatcher CLI
An app to dispatch notifications of a user to multiple destinations (such as e-mail, sms, etc).
Implementing a simple database using SQLite to store user-related data.
Currently W.I.P.

# Purpose
The purpose of this project is to learn how to implement SOLID principles (alongside Component Cohesion and Component Coupling principles).
So, overall, is a study project in order to implement Uncle Bob's architectural principles.
